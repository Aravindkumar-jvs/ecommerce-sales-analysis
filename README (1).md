# E-Commerce Sales Analysis — SQL + Python

End-to-end data analysis on 500 e-commerce transactions using Python, SQL, and Matplotlib. Covers revenue trends, customer segmentation, return rates, and city-wise performance.

## Tools Used
- Python 3
- pandas (data manipulation)
- SQLite / sqlite3 (SQL queries)
- Matplotlib (charts and visualization)

## Business Questions Answered
1. Which product categories generate the most revenue?
2. Which cities have the highest customer spend?
3. What is the monthly revenue trend over 3 years?
4. Which payment methods are most preferred?
5. How can customers be segmented by total spend?
6. Which categories have the highest return rates?

## SQL Queries Used
```sql
-- Revenue by category
SELECT Category, COUNT(*) as Orders,
       ROUND(SUM(Revenue), 2) as Total_Revenue
FROM orders WHERE Order_Status = 'Delivered'
GROUP BY Category ORDER BY Total_Revenue DESC;

-- Customer segmentation
SELECT Customer_ID, ROUND(SUM(Revenue), 2) as Total_Spend,
       CASE WHEN SUM(Revenue) > 50000 THEN 'High Value'
            WHEN SUM(Revenue) > 20000 THEN 'Mid Value'
            ELSE 'Low Value' END as Segment
FROM orders WHERE Order_Status = 'Delivered'
GROUP BY Customer_ID ORDER BY Total_Spend DESC;

-- Return rate by category
SELECT Category,
       ROUND(SUM(CASE WHEN Order_Status='Returned' THEN 1 ELSE 0 END)
       * 100.0 / COUNT(*), 1) as Return_Rate_Pct
FROM orders GROUP BY Category ORDER BY Return_Rate_Pct DESC;
```

## Key Insights
- Electronics is the top revenue category at ₹24L+
- Bangalore is the #1 city by total revenue
- Books category has the highest return rate at 15.7%
- UPI and Wallet together handle 33% of transactions
